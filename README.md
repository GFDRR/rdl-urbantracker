# Urban Data Tracker

As a part of work being done on the [The Risk Data Library](https://riskdatalibrary.org/), this Urban Data Tracker is used to analyze open dataset coverage by city and a number of fundamental dataset types. For a few examples, it can be used to determine if a city is publishing open data on housing development, administrative boundaries, and vulnerability factors related to different types of disasters.

## Based on JKAN

JKAN is a lightweight, backend-free open data portal, powered by Jekyll. We use JKAN to store sparse metadata about datasets and ready them for display in the tracker, both as individual catalog entries and in screens that are useful for analyzing coverage across the broader open data ecosystem.
